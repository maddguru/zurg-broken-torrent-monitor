## ‚ú® New Features

### 1. Unified Management UI
All torrent types in ONE interface with type badges:
- `[BRK]` Broken (Yellow)
- `[REP]` Under Repair (Cyan)
- `[BAD]` Unrepairable (Red)

### 2. Advanced Filtering
| Command | Action |
|---------|--------|
| `FB` | Filter Broken |
| `FU` | Filter Under Repair |
| `FC` | Filter Unrepairable |
| `F*` | Show All |
| `FS` | Search by name |
| `FR` | Filter by reason |
| `FX` | Clear filters |

### 3. Bulk Select by Reason
```
Enter command: BR
Enter reason pattern: infringing
Selected 8 torrent(s) matching 'infringing'
```

### 4. Live AutoRepair Toggle
Press `T` in Management to toggle AutoRepair ON/OFF instantly.

### 5. Hotkeys During Wait
```
[INFO] Next check in 30 minutes... (Press 'M' for Management, 'S' for Stats)
```
